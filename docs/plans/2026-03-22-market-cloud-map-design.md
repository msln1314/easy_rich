# 大盘云图功能设计

## 概述

实现类似 dapanyuntu.com / finviz 的市场全景可视化页面，使用 Treemap 树状图展示全市场股票，方块大小表示市值，颜色表示涨跌幅，支持多维度、多周期、分时复盘等功能。

## 数据结构

### 股票节点

```typescript
interface StockNode {
  code: string          // 股票代码
  name: string          // 股票名称
  market: string        // 市场: SH/SZ/BJ
  industry: string      // 所属行业
  value: number         // 市值(亿)
  change: number        // 涨跌幅
  pe: number           // 市盈率
  pb: number           // 市净率
  amount: number       // 成交额
  price: number        // 当前价
}
```

### 行业分组

```typescript
interface IndustryGroup {
  name: string          // 行业名称
  value: number         // 行业总市值
  change: number        // 行业涨跌幅(加权)
  children: StockNode[]
}
```

## 后端 API

### 接口定义

```
GET /stock/cloud-map/data
  - market: string (all/sh/sz/bj/kc/cy)
  - metric: string (change/pe/pb/amount)
  - period: string (today/week/month/ytd)
```

## 功能清单

### 核心功能

| 功能 | 描述 |
|------|------|
| Treemap 云图 | 方块大小=市值，颜色=涨跌幅，按行业分区 |
| 市场切换 | A股全图/上证/深证/科创板/创业板/北交所 |
| 维度切换 | 涨跌幅/市盈率PE/市净率PB/成交额 |
| 时间周期 | 今日/近1周/近1月/年初至今 |
| 个股详情弹窗 | 点击方块显示股票详情 |
| 分时复盘 | 显示当日各时点的云图快照 |
| 截图分享 | 生成图片便于分享 |

### 刷新策略

- 开盘期间：每分钟自动刷新 + 手动刷新按钮
- 收盘后：显示最终数据，仅手动刷新

## 技术选型

### 前端

- **Treemap**: ECharts Treemap 或 D3.js
- **颜色映射**: 涨(红) → 跌(绿)，涨幅越大颜色越深
- **交互**: 悬停 Tooltip + 点击详情弹窗

### 后端

- **数据源**: AKShare `stock_zh_a_spot_em()` 获取全市场数据
- **缓存**: Redis，开盘期间每分钟更新
- **快照**: 分时数据存储到 Redis
